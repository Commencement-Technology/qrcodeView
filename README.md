# react-native-qrcodeview
native view to generate and display qrcodes in native side
## Installation

```sh
npm install react-native-qrcodeview
```

## Usage

```js
import QRCodeView from "react-native-qrcodeview";

// ...

    <QRCodeView
        qrCodeValue={qrCodeValue}
        style={styles.box}
        key="QrCodeView"
        size={200}
    />
```
